# Markdown Blog (SPA на чистом JS)

Статический блог с загрузкой Markdown-статей, собственным парсером и клиентской маршрутизацией.

## Функциональность

- Загрузка списка статей из `posts/meta.json`
- Парсинг Markdown -> HTML (заголовки, жирный, курсив, код, списки, ссылки, блоки кода)
- Маршрутизация через hash (`#/post/1`, `#/favorites`)
- Кэширование статей
- Избранное (localStorage)
- Время чтения
- Автоматическое оглавление из заголовков h2

## Как запустить

1. Сохраните все файлы в указанную структуру.
2. Запустите локальный сервер: `python -m http.server 8000`
3. Откройте в браузере `http://localhost:8000`

Внимание: нужен HTTP-сервер, не открывайте index.html напрямую.
