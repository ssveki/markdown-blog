# Основы CSS: стилизация веб-страниц

**CSS** (Cascading Style Sheets) – это язык таблиц стилей, который отвечает за внешний вид веб-страницы. С его помощью можно задавать цвета, шрифты, расположение элементов и адаптивность.

## Что можно сделать с помощью CSS?

- Изменять цвет текста и фона
- Управлять отступами и размерами блоков
- Создавать анимации и переходы
- Делать сайт адаптивным (под телефоны, планшеты)

## Подключение CSS к HTML

Есть три способа:

1. **Внешний файл** (рекомендуется): `<link rel="stylesheet" href="style.css">`
2. **Внутренний стиль**: внутри тега `<style>` в `<head>`
3. **Инлайн-стиль**: атрибут `style="color: red;"`

## Пример CSS-правила

```css
/* Это комментарий в CSS */
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    margin: 0;
    padding: 20px;
}

.card {
    border: 1px solid #ccc;
    border-radius: 8px;
    padding: 16px;
    transition: transform 0.2s;
}

.card:hover {
    transform: scale(1.02);
}
